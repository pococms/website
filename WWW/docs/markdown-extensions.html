Markdown extensions

PocoCMS boasts many extensions the CommonMark standard for Markdown. They make life easier but it might be a concern moving your documents off of PocoCMS to another static site generator.

YouTube video extensions to image tags

You can get a nice YouTube video preview by pasting in the video URL after the same ![] prefix Markdown uses for inserting images. For example, this Markdown would result in the expected video shown below:

![](https://www.youtube.com/watch?v=dQw4w9WgXcQ)
