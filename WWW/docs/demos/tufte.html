Tufte theme

This gorgeous theme is derived from Tufte CSS, and it's very much in the spirity of Edward Tufte's aesthetics. Mad props to originator Dave Liepmann, who later donated his substantial work to the Tufte ecosystem.

This is an example of a classless theme, to which we added a minimal layout file to support layout elements: header, nav, and aside (right only). They tend to distract so they are suppressed in this demo.

Supported features

✅ Header on/off
✅ Nav on/off
❌ Aside left on/off
✅ Aside right on/off
❌ Footer on/off
✅ Responsive
✅ Mobile support
