Base theme

It doesn't look special, but that's the point. Base supports the entire set of theme framework features as shown in the table below. You get free support for header, nav bar, aside, footer, and mobile support. All you need to do is choose fonts, colors, padding, and so on.

It's easy to customize with minimal additions to CSS.

✅ Header on/off
✅ Nav on/off
✅ Aside left on/off
✅ Aside right on/off
✅ Footer on/off
✅ Responsive
✅ Mobile support
