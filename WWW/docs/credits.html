Most Valuable Player Award: Yusuke Inuzuka's goldmark packages

The vast majority of the work for Poco CMS was created years before Poco CMS joined the world: it's the goldmark parser and related packages. They were created by the hugely impressive Yusuke Inuzuka, a Japanese programmer whose English is better than mine. His username on GitHub is Yuin. Work like this would be impressive at any price, but free? Blows me away. And yes, I contributed.

Thanks also to the many other contributors to goldmark.

Poco CMS relies on many other inimitable Yuin, including:

  • [https://github.com/yuin/goldmark-meta](https://github.com/yuin/goldmark-meta] for handling the YAML [front matter](front-matter.html]
  • https://github.com/yuin/goldmark-highlighting to handley syntax highlighting for code

Other goldmark-related packages

  • https://github.com/abhinav/goldmark-toc for table of contents support

Alec Thomas for the Chroma highlighter

PocoCMS relies on Chroma for code syntax highlighting, thanks to the Alec Thomas.

CDN services

Amazingly, jsDelivr hosts open source files with no application process or permissions required. Just follow gh in URLs like the one below with your GitHub repository path and your file gets hosted automatically.

  ---
  Stylesheets: 
    - https://cdn.jsdelivr.net/gh/pococms/poco/pages/assets/css/poquito.css
  ---

Thanks, jsDelivr people!
