# Clerk theme

Clerk is a fun, throwback theme reminiscent of
old-school typewriters. 

## Supported features

✅ Header on/off\
✅ Nav on/off\
✅ Aside left on/off\
✅ Aside right on/off\
✅ Footer on/off\
✅ Hamburger menu\
✅ Responsive\
✅ Mobile support
